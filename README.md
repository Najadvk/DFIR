# 🕵️ DFIR Writeups | Incident Response & Forensic Investigations

This repository contains my **Digital Forensics and Incident Response (DFIR)** walkthroughs based on hands-on labs from platforms like TryHackMe, Hack The Box, and other similar environments.

The goal of this repository is to document investigations of simulated security incidents, where I analyze system activity, review artifacts, and reconstruct attacker behavior.

Each writeup represents a step-by-step investigation, focusing on understanding how attacks occur and how they can be identified through forensic analysis.


---

## Purpose

- Document walkthroughs of DFIR-related rooms and labs  
- Practice investigation and analysis skills  
- Build a collection of reports based on hands-on scenarios
